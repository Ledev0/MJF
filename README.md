## 🛠️ Prerequisites

Before getting started, make sure you have:

- **Java JDK 21** or higher installed
  - [Download Java](https://www.oracle.com/java/technologies/downloads/)
  - Verify installation: `java -version`

- **Git** installed
  - [Download Git](https://git-scm.com/)

- A code editor or IDE (Optional but recommended):
  - [IntelliJ IDEA](https://www.jetbrains.com/idea/) (recommended)
  - [VS Code](https://code.visualstudio.com/)
  - [Eclipse](https://www.eclipse.org/)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ledev0/MJF.git
   cd MJF
   ```

2. **Navigate to a chapter:**
   ```bash
   cd CH1
   ```

3. **Compile and run a project:**
   ```bash
   javac SProjects/SimpleCalculator.java
   java SProjects.SimpleCalculator
   ```

### 📌 Project Types

**SProjects** — Small, focused projects built to practice specific Java concepts. Each project targets one core idea (e.g., OOP, Collections, Generics, Exceptions).

**BProjects** — Larger, more complete projects. These are real programs that could be expanded, improved, or eventually spun off into their own standalone repos.

## ▶️ How to Run Projects

### Compile a Java file:
```bash
javac filename.java
```

### Run a Java program:
```bash
java filename
```

### Example (SimpleCalculator from CH1):
```bash
cd CH1/SProjects
javac SimpleCalculator.java
java SimpleCalculator
```

### For IDEs:
- **IntelliJ IDEA**: Open the project folder → Right-click on .java file → Run
- **VS Code**: Install Extension Pack for Java → Run button will appear
