# MJF — Master Java Fundamentals

> My personal journey to learn Java from the ground up and master its core concepts.

---

## About This Repo

This repo documents my learning path through Java fundamentals and beyond. It contains notes, exercises, and projects built while working through the language — from the basics all the way to writing real, meaningful programs.

**Primary references:**
- *Java: The Complete Reference* — Herbert Schildt
- *Effective Java* — Joshua Bloch

---

## Structure

```
MJF/
├── SProjects/   # Small practice projects — one concept at a time
└── BProjects/   # Bigger real-world projects — may grow into independent repos
```

### SProjects
Small, focused projects built to practice and reinforce specific Java concepts. Each project targets one idea (e.g. OOP, collections, generics, exceptions).

### BProjects
Larger, more complete projects. These are real programs that could be expanded, improved, or eventually spun off into their own standalone repos.
