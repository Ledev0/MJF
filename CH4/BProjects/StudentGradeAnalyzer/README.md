# StudentGradeAnalyzer

## Project Overview

This project, `StudentGradeAnalyzer`, is a simple console application designed to help analyze student grades. It utilizes Java arrays and strings to manage and process student data, such as names and their corresponding grades.

## Features

-   **Input Student Data:** Allows users to input student names and their grades.
-   **Calculate Statistics:** Computes basic statistics like average grade, highest grade, and lowest grade.
-   **Display Results:** Presents a formatted report of student grades and the calculated statistics.

## How to Run

1.  **Navigate to the project directory:**
    ```bash
    cd CH4/BProjects/StudentGradeAnalyzer
    ```
2.  **Compile the Java source file:**
    ```bash
    javac StudentGradeAnalyzer.java
    ```
3.  **Run the application:**
    ```bash
    java StudentGradeAnalyzer
    ```
