## How to Run

### Running a Single Project

**Navigate to the project:**

```bash
cd CH3/SProjects
```

**Compile the Java file:**

```bash
javac YourProjectName.java
```

**Run the program:**

```bash
java YourProjectName
```

### Example: Running CalculatorMethods

```bash
cd CH3/SProjects
javac CalculatorMethods.java
java CalculatorMethods
# Expected Output: (Example output for a calculator program using methods)
# Enter first number: 10
# Enter second number: 5
# Sum: 15
# Difference: 5
# Product: 50
# Quotient: 2
```

## 💡 Tips & Best Practices

| Tip | Explanation |
|-----|-------------|
| 🔍 **Modularize Code** | Break down complex tasks into smaller, manageable methods. |
| 📝 **Method Signatures** | Understand the components of a method signature: access modifiers, return type, name, and parameters. |
| 🧪 **Base Case (Recursion)** | Always define a clear base case for recursive methods to prevent infinite loops. |
| 🚀 **Parameter Passing** | Understand how values are passed to methods (pass-by-value). |
| 📚 **Return Values** | Ensure methods return values of the declared type. |
