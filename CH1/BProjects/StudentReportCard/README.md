# 🎓 Student Report Card

A console-based Java application that takes a student's personal information and subject scores, then generates a fully formatted report card with a final grade.

---

## 📋 Overview

This is the BProject for CH1 of the MJF journey. It brings together everything learned in Chapter 1 into one complete, real-world program.

---

## 💡 Concepts Applied

| Concept              | Usage                                      |
|----------------------|--------------------------------------------|
| Data Types           | `String`, `int`, `double` for student data |
| Arithmetic Operators | Calculating final percentage               |
| User Input (Scanner) | Taking all input from the user             |
| Printf / Println     | Formatted report card output               |
| if-else Statements   | Determining the final grade                |

---

## 📊 Grade Scale

| Percentage | Grade |
|------------|-------|
| 95 - 100   | A+    |
| 90 - 94    | A     |
| 85 - 89    | B+    |
| 80 - 84    | B     |
| 75 - 79    | C+    |
| 70 - 74    | C     |
| 65 - 69    | D+    |
| 60 - 64    | D     |
| 55 - 59    | D-    |
| 50 - 54    | E     |
| Below 50   | Fail  |

---

## ▶️ How to Run
```bash
javac StudentReportCard.java
java StudentReportCard
```
